Screen Time

Monitor and manage your device usage and screen time.

GitHub npm

Features

✓ Usage statistics
✓ App limits
✓ Downtime
✓ Content restrictions

Installation

From App Store

Open the App Store in zOS and search for "Screen Time".

Via npm

npm install @zos-apps/screen-time

How to Use

  • View daily usage
  • Set app limits
  • Schedule downtime

Development

# Install dependencies
npm install

# Start development
npm run dev

# Build for production
npm run build
